## 本地开发

每个项目都可以独立运行：

```bash
# 进入项目目录
cd calendar-app  # 或 cd snake-game

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 部署

本项目使用 Vercel 进行部署。每个子项目都会独立部署。
